# Implementacion Python

Esta implementacion ya existe en la raiz del repositorio.

## Archivos principales

- `main.py`: ejecuta los casos y genera reporte y graficas.
- `nucleo/perceptron.py`: modelo del perceptron.
- `nucleo/activaciones.py`: funciones de activacion.
- `casos/`: casos de prueba.

## Ejecucion

```bash
python main.py
```

## Nota

Para mejorar portabilidad, se recomienda migrar rutas absolutas a rutas relativas usando `pathlib`.
