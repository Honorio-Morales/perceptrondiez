# Implementacion Java (base)

Este modulo contiene una base del perceptron para completar la entrega en Java sin librerias de machine learning.

## Archivos

- `src/Perceptron.java`
- `src/Main.java`

## Estado

- Clase `Perceptron` con 6 funciones de activacion.
- Entrenamiento binario con epocas > 10.
- Casos implementados: AND, OR, spam, clima, fraude y riesgo academico.
- Salida en consola con accuracy, precision y recall por caso.

## Compilacion y ejecucion

```bash
javac src/*.java
java -cp src Main
```
