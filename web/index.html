Proyecto academico

Implementacion de un Perceptron

Implementacion multilenguaje (Python, C#, Java y JavaScript) con 6 casos de clasificacion binaria y validacion comparativa.

Que se hizo

  • Se implemento el perceptron desde cero sin librerias de machine learning.
  • Se consolidaron 6 casos: AND, OR, spam, clima, fraude y riesgo academico.
  • Se habilitaron activaciones: lineal, escalon, sigmoidal, relu, softmax y tanh.
  • Se incorporo arquitectura multilenguaje y reporte tecnico en LaTeX.

Explicacion rapida

La neurona calcula primero un valor lineal z = w1*x1 + w2*x2 + b. Luego aplica la funcion de activacion elegida para obtener una salida continua o discreta. Finalmente se umbraliza para clasificacion binaria.

Esta simulacion no reemplaza el entrenamiento completo, pero ayuda a comprender como influyen pesos y sesgo en la decision final.
